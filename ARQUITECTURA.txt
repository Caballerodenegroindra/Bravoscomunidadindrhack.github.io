FLUJO DE DATOS
==============

1. Usuario interactúa con HTML
    ↓
2. JavaScript maneja el evento
    ↓
3. Valida datos (utilities.js)
    ↓
4. Llama a Firebase (config.js)
    ↓
5. Firebase procesa la solicitud
    ↓
6. Retorna datos
    ↓
7. JavaScript actualiza el DOM
    ↓
8. Usuario ve cambios

ALMACENAMIENTO
==============

- localStorage: Tokens, datos de usuario
- Firestore: Usuarios, cursos, comentarios
- Storage: Imágenes, documentos PDF

SEGURIDAD
=========

1. Contraseñas hasheadas
2. Tokens en localStorage
3. Verificación de rol en cada acción
4. Validaciones en cliente y servidor
5. No exponer datos sensibles
6. CORS configurado

NOTAS IMPORTANTES
=================

1. Todas las páginas deben incluir:
   - Barra de navegación
   - Footer
   - Enlaces a CSS global y específico
   - Scripts en el orden correcto

2. Responsive design en todas las páginas:
   - Mobile (< 480px)
   - Tablet (480-768px)
   - Desktop (> 768px)

3. Sistema de autenticación:
   - localStorage para tokens
   - Estado pendiente por defecto
   - Solo admin puede aprobar

4. Panel administrador:
   - Ruta privada
   - Verificación de rol admin
   - CRUD completo de usuarios y cursos

5. Firebase:
   - Reemplazar credenciales en config.js
   - Crear colecciones: users, courses, approvals
   - Configurar Storage para imágenes
